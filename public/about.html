Project Description

For this project, we created a single page application about crimes in St. Paul Minnesota. The project consisted of the Vue framework and 3 APIs (Leaflet, Nominatim, and our own API). This single page application will allow users to traverse the map and see the different crimes update in the table.

Interesting Findings

  1. Proactive Police Visits were one of the most frequent incident types.
  2. There were not many homocides listed within the database for the large range of time.
  3. 2018-2020 had very similar numbers of Rape, despite COVID forcing people to quarantine.
  4. One of the biggest crimes during the start of COVID (March-June 2020) were mostly Theft and Auto Theft.
  5. Capital River had the most Graffiti crime in September 2021.
  6. There are only 3 homicide crimes in the Summit/University neighborhood since 2017.
